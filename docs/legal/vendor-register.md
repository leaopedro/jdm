# Vendor Register

Canonical vendor-onboarding and annual-review register for JDM Experience.
This file satisfies the hard onboarding gate in
`docs/legal/vendor-management-policy.md`.

Usage rules:

- Add or update a row before introducing any new vendor, SDK, hosted runtime,
  webhook integration, or service account.
- Keep unresolved legal, contract, or security facts explicitly marked as
  `Pending`.
- Re-review every approved vendor at least once every 12 months or whenever the
  integration scope expands.

Scope note:

- This register captures the 12 vendors already enumerated in `LGPD_scan.md`
  section 7.
- Several vendors predate the existence of this formal register. Their
  `Onboarding date` below records when the register row was first formalized for
  governance, not necessarily the original production-adoption date.
